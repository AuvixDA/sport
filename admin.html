Настройки Telegram

Токен бота, который прислал @BotFather после команды /newbot.
Без «@» и без «t.me/» — только сам username, например pulsetreners (получится ссылка t.me/pulsetreners).
Именно числовой Telegram ID тренера (не username!). Чтобы узнать: тренер пишет любое сообщение боту @userinfobot, тот в ответ присылает «Id: 123456789» — это значение и нужно сюда вписать.

Важно: после этого тренер должен ещё написать что‑нибудь (например «привет») своему собственному боту (тому, чей токен указан выше) — иначе Telegram не разрешит боту написать тренеру первым, и уведомления не будут доходить.
